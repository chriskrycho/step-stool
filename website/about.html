A(nother) static site generator.
(This one is in Python.)

Step Stool is a simple static site generator. You write your content in Markdown, run a quick command line script, and it gets published to the web. No content management system, no database, just simple HTML files.

Step Stool supports custom templates (via Jinja2), so you can make your site look pretty much however you want. You can apply different templates to different sections of your site, using a single line in a configuration file.

Step Stool also allows you to structure your site pretty much however you choose. If you want to have nested categories, go for it. If you want an entirely flat structure with an ever-growing list of tags, you can do that, too.
